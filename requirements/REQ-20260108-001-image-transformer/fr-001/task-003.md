# Task 003: Create appsettings.json Configuration File

Create the appsettings.json file with default configuration values for the application.

## Deliverable
- appsettings.json file created in the project root with keys:
  - InputFolder: path to input directory.
  - OutputFolder: path to output directory.
  - PngSizeThresholdMB: size threshold for PNG conversion in MB (default 5).
  - JpegTargetSizeMB: target size for JPEG resizing in MB.
  - ResizeStepPercentage: percentage to reduce dimensions per resize step.
- File set to copy to output directory.
